# nutzcn客户端

采用[weex](https://weex-project.io/)实现的[nutzcn论坛](https://nutz.cn/)客户端

# 开发计划

一期分为三个阶段：

1. 挖坑与基本框架搭建，内部小范围测试
2. 用户登陆发帖通知消息等基本功能，测试版上线论坛用户加入测试
3. 完成剩余功能，正式版发布
